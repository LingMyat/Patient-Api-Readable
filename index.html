Api Patient Panel

Welcome to the Appointment API, a RESTful API for Hospital Appointment e-commerce web application.

This API is developed with PHP(Laravel) and MySQL.

Nrc Prefix

Get All Nrc Prefixs | GET http://127.0.0.1:8000/api/v1/patient/nrc
Description
Fetch all Nrc prefixs from the database.
Get All Nrc Prefix Codes | GET http://127.0.0.1:8000/api/v1/patient/nrc/code
Description
Fetch all Nrc prefix codes from the database.
Get Nrc Name_MM With Nrc_Code | GET http://127.0.0.1:8000/api/v1/patient/nrc/{nrc_code}/name
Description
Fetch Nrc Prefix name_mm by nrc_code from the database. Nrc-code can get from "Get All Nrc Prefix Codes" .
Example
"http://127.0.0.1:8000/api/v1/patient/nrc/9/name"

Auth

Register | POST http://127.0.0.1:8000/api/v1/patient/auth/register
Description
Patient register.
Body
{
"name": "User" ,
"email_or_phone": "user@gmail.com" ,
"password": "password" ,
"confirm_password": "password"
}
Login | POST http://127.0.0.1:8000/api/v1/patient/auth/login
Description
Patient login.
Body
{
"email_or_phone": "user@gmail.com" ,
"password": "password"
}
Logout | DELETE http://127.0.0.1:8000/api/v1/patient/auth/logout
Description
Patient Logout.To perform this action, user must be authenticated.

Description

Doctors

Get All Doctors | GET http://127.0.0.1:8000/api/v1/patient/doctors
Description
Fetch all doctors from the database.
Get Specific Doctors | GET http://127.0.0.1:8000/api/v1/patient/doctors?doctor_id=1
Description
Fetch doctor by ID from the database.
Get Doctors With Diagnosis | GET http://127.0.0.1:8000/api/v1/patient/doctors?diagnosis_id={diagnosis_id}
Description
Fetch specific doctors by diagnosis ID from the database.Diagnosis_id can get from "Get All Diagnosis"
Body
"http://127.0.0.1:8000/api/v1/patient/doctors?diagnosis_id=9"

Chat Room

Get All Chat Rooms | GET http://127.0.0.1:8000/api/v1/patient/rooms
Description
Fetch all chat rooms from the database.
Get All Chat Messages From Specific Rooms | Get http://127.0.0.1:8000/api/v1/patient/rooms/{room_id}/room-messages
Description
Fetch all messages from specific_rooms. Room_id can get from "Get All Chat Rooms" .
Example
"http://127.0.0.1:8000/api/v1/patient/rooms/1/room-messages"
Send Message | POST http://127.0.0.1:8000/api/v1/patient/rooms/store-room-message
Description
Send message to room. Store messages.
Body
{
"room_id": 1 ,
"message": "Lorem Ipsum is simply dummy text of the printing and typesetting industry."
}
Send Image | POST http://127.0.0.1:8000/api/v1/patient/rooms/store-chat-image
Description
Send image to room. Store chat image.
Body
{
"room_id": 1 ,
"image": "something.jpg / something.png"
}
