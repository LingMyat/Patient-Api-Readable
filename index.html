Api Patient Panel

This API is developed with PHP(Laravel) and MySQL.

Nrc Prefix

Get All Nrc Prefixs | GET http://127.0.0.1:8000/api/v1/patient/nrc
Description
Fetch all Nrc prefixs from the database.
Get All Nrc Prefix Codes | GET http://127.0.0.1:8000/api/v1/patient/nrc/code
Description
Fetch all Nrc prefix codes from the database.
Example
[
1 ,
2 ,
3 ...
]
Get Nrc Name_MM With Nrc_Code | GET http://127.0.0.1:8000/api/v1/patient/nrc/{nrc_code}/name
Description
Fetch Nrc Prefix name_mm by nrc_code from the database. Nrc-code can get from "Get All Nrc Prefix Codes" .
Example
"http://127.0.0.1:8000/api/v1/patient/nrc/9/name"

Auth

Register | POST http://127.0.0.1:8000/api/v1/patient/auth/register
Description
Patient register.
Body
{
"name": "User" ,
"email_or_phone": "user@gmail.com" ,
"password": "password" ,
"confirm_password": "password"
}
Login | POST http://127.0.0.1:8000/api/v1/patient/auth/login
Description
Patient login.
Body
{
"email_or_phone": "user@gmail.com" ,
"password": "password"
}
Get Profile | GET http://127.0.0.1:8000/api/profile
Description
This API will return current Token owner. To perform this action, user must be authenticated.
Update | POST http://127.0.0.1:8000/api/v1/patient/auth/update
Description
Patient Profile Update. To perform this action, user must be authenticated.
Body
{
"name": "User Name" ,
"email": "user@gmail.com" ,
"phone": "09123456789" ,
"address": "user address" ,
"date_of_birth": "YY-mm-dd" ,
"gender": "Male or Female" ,
"nrc_number": "User Nrc Number (6-number)" ,
"nrc_code": "Nrc_code From Get All Nrc Prefix Codes API" ,
"nrc_name": "Nrc_name From Get Nrc Name_MM With Nrc_Code API" ,
}
Logout | DELETE http://127.0.0.1:8000/api/v1/patient/auth/logout
Description
Patient Logout.To perform this action, user must be authenticated.

Diagnosis

Main Diagnosis
Get All Main Diagnosis | GET http://127.0.0.1:8000/api/v1/main-diagnosis
Description
Fetch all main diagnosis from the database. !! Doctor Specilist are only sub_diagnosis !!
Example
{
"id": 1 ,
"name": "ကင်ဆာရောဂါများ" ,
"sub_diagnosis": [
{
"id": 8 ,
"name": "Can Cer" ,
"image": "something.jpg / something.png"
}
]
}
Get Specific Main Diagnosis | GET http://127.0.0.1:8000/api/v1/main-diagnosis?id={id}
Description
Fetch main diagnosis by ID from the database.
Body
"http://127.0.0.1:8000/api/v1/main-diagnosis?id=6"

Chat Room

Get All Chat Rooms | GET http://127.0.0.1:8000/api/v1/patient/rooms
Description
Fetch all chat rooms from the database.
Example
{
"id": 1 ,
"name": "Traditional Medicine" ,
"room_img": "something.jpg / something.png"
}
Get All Chat Messages From Specific Rooms | Get http://127.0.0.1:8000/api/v1/rooms/{room_id}/room-messages
Description
Fetch all messages from specific_rooms. Room_id can get from "Get All Chat Rooms" . To perform this action, user must be authenticated.
Example
"http://127.0.0.1:8000/api/v1/rooms/1/room-messages"
{
"room": {
"id": 1 ,
"name": "Traditional Medicine" ,
"room_img": "something.jpg / something.png"
} ,
"messages": [
{
"id": 1 ,
"message": "Message Text(can be long text)" ,
"sender": {
"id": 1 ,
"name": "Ling Myat Aung" ,
"image": "something.jpg / something.png" ,
"role": "can be (Patient or Doctor)"
} ,
"child_messages": [
{
"id": 2 ,
"message": "Message Text(can be long text)"
}
]
}
] ,
"last_sender": {
"id": 1 ,
"name": "Ling Myat Aung" ,
"image": "something.jpg / something.png" ,
"role": "can be (Patient or Doctor)"
}
}
Send Message | POST http://127.0.0.1:8000/api/v1/rooms/store-room-message
Description
Send message to room. Store messages. To perform this action, user must be authenticated.
Body
{
"room_id": 1 ,
"message": "Lorem Ipsum is simply dummy text of the printing and typesetting industry."
}
Send Image | POST http://127.0.0.1:8000/api/v1/rooms/store-chat-image
Description
Send image to room. Store chat image. To perform this action, user must be authenticated.
Body
{
"room_id": 1 ,
"image": "something.jpg / something.png"
}
